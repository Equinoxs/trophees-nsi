# `Vector2` - Représentation des vecteurs et points en 2D
## Attributs
- `x` : *`int`*
- `y` : *`int`*

## Méthodes
- `__init__(x=0, y=0)` \
  Initialise les coordonnées x et y. \
  Paramètres :
  * `x` : *`float`*
  * `y` : *`float`*
- `copy(position=None)` \
  Renvoie une copie de cette instance ou prend les valeurs d'une autre instance si elle est fournie en paramètre. \
  Paramètre :
  * `position` : *`Vector2`*
